Erlang-C-Modelle

Mit Hilfe der Erlang-C-Formel lassen sich in einfachen Warteschlangenmodellen die relevanten Kenngrößen exakt berechnen. "Einfach" bedeutet dabei, dass das Modell wie folgt aufgebaut sein muss:

  • Die Zwischenankunftszeiten sind exponentiell verteilt.
  • Die Bedienzeiten sind exponentiell verteilt.
  • Es existiert eine feste Anzahl von Bedienern.

Die klassische Erlang-C-Formel kann Ungeduld der Kunden nicht abbilden, sehr wohl jedoch die erweiterte Erlang-C-Formel, so dass außerdem folgende Eigenschaft in einem einfachen Modell vorherrschen darf:

  • Optional sind die Kunden nur begrenzt lange bereit zu warten. Ist dies der Fall, so muss die Wartezeittoleranz durch die Exponentialverteilung beschrieben werden können.

Ein entsprechendes Modell wird dann als M/M/c bzw. M/M/c/unendlich+M Modell bezeichnet.

Allen-Cunneen-Näherungsformel

Die Allen-Cunneen-Näherungsformel lässt eine Lockerung der im Erlang-C-Modell definierten Einschränkungen zu: Statt exponentiell verteilten Zwischenankunfts- und Bedienzeiten werden jetzt beliebige Verteilungen, die durch ihren Erwartungswert und ihren Variationskoeffizient charakterisiert sind, zugelassen. Dafür muss die Exaktheit aufgegeben werden, d.h. selbst bei einem Modell, welches alle Vorgaben der Allen-Cunneen-Näherungsformel einhält, muss mit Abweichungen zwischen Näherungsformel-Ergebnis und Simulationsrealität gerechnet werden.
