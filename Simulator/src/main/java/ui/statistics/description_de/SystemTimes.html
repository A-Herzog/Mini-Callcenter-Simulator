Auf dieser Statistikseite werden Informationen zu den Verweilzeiten der Kunden ausgegeben. Es wird dabei zwischen den Verweilzeiten der erfolgreichen Kunden und den Verweilzeiten der Warteabbrecher (d.h. den Abbruchzeiten) unterschieden. Für erfolgreiche Kunden ist die Verweilzeit als Summe aus Warte- und Bedienzeit. Sowohl für erfolgreiche als für unerfolgreiche Kunden werden jeweils Mittelwert, Standardabweichung, Variationskoeffizient und die Quantilen der Zeitdauern ausgegeben.
