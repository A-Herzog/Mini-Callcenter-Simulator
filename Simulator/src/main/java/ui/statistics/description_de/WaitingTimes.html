Auf dieser Statistikseite werden Informationen zu den Wartezeiten der Kunden ausgegeben. Es wird dabei zwischen den Wartezeiten der erfolgreichen Kunden und den Wartezeiten der Warteabbrecher (d.h. den Abbruchzeiten) unterschieden. In beiden Fällen werden jeweils Mittelwert, Standardabweichung und Variationskoeffizient ausgegeben.

Service-Level

Zusätzlich wird der Service-Level an der 20-Sekunden-Schwelle ausgegeben. Dieser Wert gibt an, wie hoch der Anteil an Kunden ist, der höchstens 20 Sekunden warten musste.

Da der Service-Level eng verwandt mit den Quantilen ist, werden auch die Quantile der Wartezeiten in diesem Bereich ausgegeben.
