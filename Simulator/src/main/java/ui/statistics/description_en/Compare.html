Erlang-C formula

By using the Erlang-C formula the relevant performance indicators can be calculated exactly. In this context "simple" means the model needs to have the following properties:

  • The inter-arrival times are exponential distributed.
  • The service times are exponential distributed.
  • There is a fixed number of operators.

The classical Erlang-C formula cannot represent clients impatience. But the extended Erlang-C formula can represent impatience, so the following property may also prevail in a simple model:

  • Optionally, clients are only willing to wait for a limited time. If this is the case, it waiting time tolerance has to be exponential distributed.

A corresponding model is then called M/M/c or M/M/c/ infinity+M model.

Allen-Cunneen approximation formula

The Allen-Cunneen approximation formula allows a relaxation of the restrictions defined in the Erlang-C model: Instead of exponentially distributed inter-arrival and service times, any distributions characterized by their expected value and coefficient of variation are now allowed. For this, the absolute accuracy has to be given up, i.e. even with a model which complies with all specifications of the Allen-Cunneen approximation formula, differences between the approximation formula results and the simulation reality have to be expected.
