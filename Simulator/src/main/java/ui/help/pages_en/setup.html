Settings

The settings dialog consists of 2 dialog pages, which are used to configure the operating parameters of the simulator:

User interface

On this dialog page the appearance and behavior of the simulators user interface can be set:

Program language

In the field can be select whether the program interface should be displayed in English or in German.
(If German is selected as the system language, the default value is "German", otherwise it is "English".)

Window size on program start

In this selection field, you can set how the program window should look at the start of the simulator. In addition to "Default window size" and "Full screen", the option "Restore last window size" provides the option to save the window size at the end of the program and restore it at the next startup.
(Default value for this option: "Default window size".)

Graphics

Resolution when saving images

In this field can be specified which resolution should be used in pixels both in the horizontal and in the vertical direction when storing graphics from the statistics results as pixel graphics.
(Default value is 1000 pixels.)

Statistics

Open texts and tables

In this selection fields you can set up which external programs should be offered for transferring the data from the statistics text and table pages.
(As default Word and Excel are selected.)
