Simulation results

The statistics viewer is displayed directly after the end of a simulation run. It displays all data collected during the simulation process.

On the left side of the window you can see a tree structure that lists all available topics. If you click on one of these topics, the data for this topic appears on the right side of the window.

Using the toolbar button Model for these results, the call center model on which the statistics data are based can be displayed and loaded back into the editor.
