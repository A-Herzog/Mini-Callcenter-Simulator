Allgemeines

Der Name und die Beschreibung dienen nur zur späteren Identifikation des Modells und haben keine Auswirkungen auf die Simulation. Bei beiden handelt es sich um Freitextfelder.

Für die Anzahl an zu simulierenden Ankünften gilt, dass eine höhere Anzahl zu statistisch stabileren Ergebnissen führt, aber gleichzeitig auch die Simulationslaufzeit verlängert.

Als Einschwingphase kann optional eine Anzahl an Anrufen angegeben werden, die zusätzlich vorab simuliert werden sollen. Erst nach dem Abschluss der Simulation dieser Anrufer startet die Statistikaufzeichnung und die eigentlich angegebene Anzahl an Anrufern wird simuliert. Auf diese Weise können Randeffekt, bedingt dadurch, dass die Simulation mit einem leeren System startet, vermieden werden.
