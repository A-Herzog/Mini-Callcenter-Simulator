Modell

Auf dieser Seite wird das generelle Aussehen des abgebildeten Simulationsmodells dargestellt:

  • Kunden treffen ein und stellen sich an die Warteschlange an - oder werden abgewiesen, wenn in der Warteschlange kein Platz mehr ist.
  • Wartende Kunden geben das Warten evtl. auf, wenn ihre persönliche Wartezeittoleranz überschritten wird.
  • Kunden, die die Warteschlange verlassen ohne bedient worden zu sein, starten evtl. später einen weiteren Versuch.
  • Wird ein Bediener frei, so wird die Bedienung des jeweils nächsten wartenden Kunden gestartet.
  • Nach eine Bedienung geht ein Bediener evtl. in eine zusätzliche Nachbearbeitungszeit, bevor er sich wieder als verfügbar meldet.
  • Kunden verlassen nach der Bedienung das System oder werden - wenn ihr Anliegen noch nicht vollständig geklärt wurde - an die Warteschlange zurück verwiesen.

Auf dieser Dialogseite sind keine Einstellmöglichkeiten vorhanden.
