Bedienungen

Die Anzahl an Agenten gibt an, wie viele Bediener gleichzeitig an der Bedienstation arbeiten.

Die Bedienreihenfolge gibt an, welcher der wartenden Kunden jeweils als nächstes bedient werden soll. Die Bedienung kann in Ankunftsreihenfolge (FIFO) oder in umgekehrter Ankunftsreihenfolge (LIFO) erfolgen. Eine FIFO-Bedienung entspricht dem Prinzip einer Warteschlange. Eine LIFO-Bedienung tritt immer dann auf, wenn die wartenden Werkstücke gestapelt werden und der Stapel nur von oben befüllt und Werkstücke wieder entnommen werden können.

Im klassischen Callcenter-Bereich spricht jeweils ein Agent mit jeweils einem Kunden. In anderen Bereichen kann jedoch der Fall eintreten, dass jeweils mehrere Kunden pro Bedienung bedient werden.

Über die Bedienzeitverteilung werden die individuellen Bedienzeiten der Kunden bestimmt. Während bei der Zwischenankunftszeiten häufig die Exponentialverteilung passend war, sind Bedienzeiten eher selten exponentiell verteilt. Häufiger anzutreffen sind hier die Lognormal- oder die Gamma-Verteilung.

Nach einem Gespräch kann es nötig sein, den Kunden zu einem weiteren Agenten zu leiten, um sein Anliegen abschließend zu behandeln. Welcher Anteil der bedienten Anrufer nach einem Gespräch weitergeleitet werden soll, kann über das Eingabefeld Weiterleitungswahrscheinlichkeit festgelegt werden.
