Ankünfte

Treffen die Kunden einzelnen ein (was den Normalfall in einem Callcenter darstellt), so kann der Wert in dem Feld Kunden pro Ankunft auf 1 belassen werden. Treffen jeweils mehrere Kunden gleichzeitig ein, so kann hier auch ein höherer Wert eingetragen werden.

Über die Verteilung der Zwischenankunftszeiten wird festgelegt, mit welchem Abstand die Kunden eintreffen. Neben dem mittleren Abstand der Kunden steuert die Verteilung auch die Streuung der Ankünfte. Treffen die Kunden unabhängig ein, so ist die Exponentialverteilung aufgrund ihrer Gedächtnislosigkeit die richtige Wahl.
