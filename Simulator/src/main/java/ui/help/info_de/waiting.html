Warteraum und Wartezeittoleranz

Wird die Größe des Warteraums begrenzt, so werden Kunden, die eintreffen während alle Warteplätze belegt sind, abgewiesen. Kunden, die bereits bedient werden, belegen keinen Platz in der Warteschlange, d.h. eine Warteraumbegrenzung bezieht sich nur auf die wartenden Kunden.

Über die Wartezeittoleranz kann gesteuert werden, ob die Kunden bereit sind, beliebig lange zu warten, oder ob sie das Warten irgendwann aufgeben. Die Wartezeittoleranz ist dabei kein für alle Kunden konstanter Wert, sondern wird individuell pro Kunde gemäß der eingestellten Verteilung ermittelt.

Die Erfassung der Autokorrelation der Wartezeiten verlangsamt die Simulation, ermöglicht aber zu sehen, wie stark der Einfluss der Wartezeit eines Kunden auf die Wartezeiten der folgenden Kunden ausfällt.
