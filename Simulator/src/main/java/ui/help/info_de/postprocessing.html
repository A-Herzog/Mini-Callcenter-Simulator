Nachbearbeitungszeiten

Die Nachbearbeitungszeiten geben die Zeitdauern, die ein Agent nach einem Gespräch mit einem Kunden benötigt, bevor er sich wieder als verfügbar meldet, an. Wird als Nachbearbeitungszeitenverteilung die Ein-Punkt-Verteilung mit Erwartungswert 0 gewählt, so treten keine Nachbearbeitungszeiten auf.
