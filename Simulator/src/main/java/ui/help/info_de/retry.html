Wiederholungen

Ein Kunde, der entweder aufgrund der Tatsache, dass alle Warteplätze belegt waren, abgewiesen wurde oder aber aufgrund einer zu kurzen individuellen Wartezeittoleranz das Warten vorzeitig aufgegeben hat, kann später einen weiteren Anrufversuch starten. Über das Eingabefeld Wiederholwahrscheinlichkeit kann festgelegt werden, mit welcher Wahrscheinlichkeit dies auftritt.

Wenn ein Kunden sich entschließt, später einen weiteren Anrufversuch zu starten, so wartet er eine gewisse Zeit, bis er den weiteren Anrufversuch tätigt. Über den Verteilungseditor zur Festlegung der Wiederholabstände lässt sich einstellen, gemäß welcher Wahrscheinlichkeitsverteilung die individuellen Wiederholabstände festgelegt werden sollen.
