Simulationsergebnisse

Der Statistik-Betrachter wird direkt nach dem Ende eines Simulationslaufes angezeigt. Er stellt alle im Simulationsprozess erhobenen Daten dar.

Auf der linken Seite des Fensters ist eine Baumstruktur zu sehen, in der alle verfügbaren Themen aufgelistet wird. Wird eines dieser Themen angeklickt, so erscheinen die Daten zu diesem Thema auf der rechten Fensterhälfte.

Über die Symbolleistenschaltfläche Modell zu diesen Ergebnissen lässt sich das Callcenter-Modell, welches diesen Statistik-Daten zugrunde liegt anzeigen und auch wieder in den Editor laden.
