Inhalt der Mini Callcenter Simulator Hilfe

Um direkt Hilfe zu der jeweils aktiven Funktion zu erhalten, können Sie auch jederzeit im Programm die F1-Taste drücken.

Themen in dieser Hilfe

  • Modell-Editor
  • Simulationsergebnisse
  • Modelle vergleichen
  • Einstellungen
  • Warteschlangenrechner
  • Kommandzeilenbefehl ausführen

Statistik-Kenngrößen

  • Mittelwerte
  • Standardabweichungen und Varianzen
  • Variationskoeffizienten
  • Quantile

Weitere Ressourcen

  • DTD-Datei zur Beschreibung der XML-Formate
  • XSD-Datei zur Beschreibung der XML-Formate
