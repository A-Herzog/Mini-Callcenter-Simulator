Einstellungen

Der Einstellungen-Dialog besteht aus 4 Dialogseiten, über die die wesentlichen Betriebsparameter des Simulators eingestellt werden können:

Benutzeroberfläche

Auf dieser Dialogseite kann das Aussehen und Verhalten der Benutzeroberfläche des Simulators eingestellt werden:

Anzeigesprache

In dem Feld kann ausgewählt werden, ob die Programmoberfläche auf Deutsch oder auf Englisch angezeigt werden soll.
(Ist als Systemsprache Deutsch gewählt, so wird als Vorgabewert "Deutsch" eingestellt, sonst "Englisch".)

Fenstergröße beim Programmstart

Der Simulator kann beim Programmstart als normales Fenster oder als Vollbildfenster angezeigt werden. Wenn Sie möchten, dass der Simulator beim Starten direkt als Vollbildfenster angezeigt wird und Sie ihn nicht erst maximieren müssen, damit er den gesamten Bildschirmplatz einnimmt, so können Sie diese Option von "Normales Fenster" auf "Vollbild" umstellen. Außerdem steht die Option "Letzte Größe wiederherstellen" zur Auswahl, die dafür sorgt, dass die momentane Fenstergröße und -position beim Beenden gespeichert wird und beim nächsten Start des Simulators diese Einstellungen wiederhergestellt werden.
(Vorgabewert ist "Normales Fenster".)

Grafiken

Auflösung beim Speichern

Auf dieser Dialogseite kann festgelegt werden, welche Auflösungen beim Speichern von Diagramme als Pixel-Grafiken verwendet werden soll. Die in dem Eingabefeld Auflösung beim Speichern angegebene Zahl gibt die Höhe und Breite in Pixeln an.
(Vorgabewert sind 1000 Pixel.)

Simulation

Alle CPU-Kerne verwenden

Ist diese Option aktiviert, so wird die Simulation auf alle verfügbaren CPU-Kerne verteilt. Ist sie deaktiviert, so wird nur ein CPU-Kern genutzt, d.h. die Simulation läuft langsamer ab.
(Als Vorgabe ist diese Option aktiviert.)

Statistik

Texte und Tabellen öffnen

Über diese Auswahlfelder kann festgelegt werden, welche externen Programme auf den Statistik-Text- und -Tabellen-Seiten zur Übernahme der Daten angeboten werden sollen.
(Als Vorgabe sind Word und Excel ausgewählt.)
